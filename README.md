# Superstore Sales Data Analysis

## Project Overview
This project analyzes Superstore sales data to uncover trends in sales performance, profitability, and product demand.  
The workflow includes data cleaning and preparation in Python, followed by interactive data visualization in Power BI.

The goal of this project was to practice an end-to-end data analysis process: transforming raw data into meaningful business insights.

---

## Tools & Technologies
- **Python** (pandas)
- **Microsoft Excel**
- **Power BI Desktop**

---

## Data Cleaning & Preparation (Python)
The raw Superstore CSV file was cleaned and prepared using Python:
- Rounded and standardized sales-related numeric values
- Converted date columns into proper datetime formats
- Ensured consistency for downstream analysis
- Exported the cleaned dataset to Excel for Power BI ingestion

The Python script used for data cleaning is included in this repository.

---

## Power BI Dashboard
An interactive dashboard was built in Power BI Desktop to visualize key business metrics.

### Dashboard Features:
- **Total Sales**
- **Total Profit**
- **Profit Margin**
- **Total Orders**
- Sales trend over time (line chart)
- Most profitable products (bar chart)
- Most ordered products (bar chart)
- Top 10 most profitable products (table)
- Interactive slicers:
  - Time period
  - Category
  - Region
